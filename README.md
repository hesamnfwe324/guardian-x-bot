- Anti-Mention Spam, Anti-Username Spam, Anti-Forward Spam
- Anti-Emoji Spam, Anti-Hashtag Spam, Anti-Phone Numbers
- Anti-Scam Links, Anti-Crypto Scam, Anti-NSFW Words
- Captcha System (Button/Math/Question)
- Full Lock System (18 content types)

## 🚀 Quick Start

### Prerequisites
- Python 3.12+
- PostgreSQL 16+
- Redis 7+
- Bot Token from [@BotFather](https://t.me/BotFather)

### Local Development

```bash
# Clone and install
git clone <repo>
cd guardian-bot
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your credentials

# Run migrations
python scripts/migrate.py

# Start in polling mode (development)
BOT_TOKEN=your_token WEBHOOK_URL="" DATABASE_URL=... REDIS_URL=... python -m bot.main
```

### Docker Compose

```bash
cp .env.example .env
# Fill in BOT_TOKEN and BOT_OWNER_ID

docker-compose up -d
```

## ☁️ Deploy to Render.com

### Method 1: render.yaml (Recommended)

1. Push your code to GitHub
2. Connect the repo on [Render.com](https://render.com)
3. Render will auto-detect `render.yaml`
4. Set the required environment variables:
   - `BOT_TOKEN` — Your Telegram bot token
   - `BOT_OWNER_ID` — Your Telegram user ID

That's it! Render will provision PostgreSQL, Redis, and deploy the bot automatically.

### Method 2: Manual Setup

1. Create a **Web Service** on Render
   - Runtime: Docker
   - Health check path: `/health`
2. Create a **PostgreSQL** database
3. Create a **Redis** instance
4. Set environment variables from `.env.example`

### Required Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `BOT_TOKEN` | ✅ | From @BotFather |
| `DATABASE_URL` | ✅ | PostgreSQL connection string |
| `REDIS_URL` | ✅ | Redis connection string |
| `WEBHOOK_URL` | ✅ (prod) | Your app's public URL |
| `BOT_OWNER_ID` | ✅ | Your Telegram user ID |
| `WEBHOOK_SECRET` | Auto | Generated by Render |
| `SENTRY_DSN` | ❌ | Optional error tracking |

## 📋 Commands Reference

### Moderation (Admin only)
| Command | Description |
|---------|-------------|
| `/ban @user [reason]` | Permanently ban a user |
| `/unban @user` | Unban a user |
| `/kick @user [reason]` | Remove from group |
| `/mute @user [time] [reason]` | Restrict messaging (e.g. `1h`, `30m`) |
| `/unmute @user` | Remove mute |
| `/warn @user [reason]` | Issue a warning |
| `/unwarn @user` | Remove latest warning |
| `/warns @user` | View warning history |
| `/note @user <text>` | Add an admin note |
| `/history @user` | View full user history |

### Economy
| Command | Description |
|---------|-------------|
| `/balance` | Check your wallet |
| `/daily` | Claim daily reward |

### Duels
| Command | Description |
|---------|-------------|
| `/duel @user [wager]` | Challenge a user (reply or @mention) |

### General
| Command | Description |
|---------|-------------|
| `/start` | Start the bot (private chat) |
| `/menu` | Open main menu |
| `/help` | Help center |

- `REFERRAL_REWARD` = 150 coins
- `ACTIVITY_XP_PER_MSG` = 2 XP
- `MAX_WARN_COUNT` = 3 warnings
